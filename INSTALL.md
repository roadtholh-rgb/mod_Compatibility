# Step install

- Extract folder epic_fight_villages_guard-1.1-forge-1.20.1.jar.
- Copy folder data in patch/data to copy epic_fight_villages_guard-1.1-forge-1.20.1/copy_here
- Copy mods.toml to epic_fight_villages_guard-1.1-forge-1.20.1/META-INF/copy_here.
- Finally, create the file `nama.jar` by Selecting the following files in this list :
  assets
  data
  META-INF
  net
  pack.mcmeta
  
 - Place the mod file in your mods folder.
 - And don't forget to install the resource pack listed in the README.md.
